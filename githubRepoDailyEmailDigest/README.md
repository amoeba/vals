# GitHub Repo Daily Email Digest

Sends you a daily email with a digest of the issue and pull request activity
(opened, updated, closed) for a set of GitHub repositories.

## Setup

1. Set the `GITHUB_TOKEN` environment variable with your GitHub personal access token
2. Set the `GITHUB_REPOS` environment variable with a comma-separated list of repositories in the format `owner/repo`

Example:
```
GITHUB_REPOS=apache/arrow,apache/arrow-adbc,apache/arrow-go,apache/arrow-js,apache/arrow-dotnet
```
